# Lab Safety

CyberLab Toolkit is designed for safe cybersecurity education.

## Allowed Use

You may use this project for:

- Local lab practice
- University projects
- Authorized testing
- Defensive security learning
- Portfolio demonstrations

## Not Allowed

Do not use this project for:

- Attacking real systems
- Brute forcing real services
- Credential theft
- Phishing
- Malware development
- Unauthorized scanning
- Denial of service
- Exploitation without permission

## Brute Force Simulator Boundary

The brute-force simulator module will only work against a local demo authentication function.

It will not support real SSH, FTP, email, website, or external login brute forcing.
